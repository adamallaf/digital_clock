# digital_clock
A 4x7-seg digital clock project to replace the circuit in a clock i bought on Aliexpress

## Hardware:
 * 4x7-seg display
 * RTC on I2C - (DS1337?)
 * Temperature sensor on I2C (FM75?)
 * Alarm buzzer
 * Light sensor used to set the display brightness (LDR)
 * 2 control buttons
